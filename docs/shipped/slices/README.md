# Shipped Slices

Completed slice documents land here. When a slice in `docs/slices/` reaches `SHIPPED`
status (fully validated, in production), move its document into this directory per the
completion rule in `docs/documentation.md`.

This keeps `docs/slices/` scoped to active, planned, and partial work, and preserves a
clean archive of delivered slices.

Empty until the first slice ships.
